While Loop Assignment Explanation

This program demonstrates the use of while loops in Python.

1. The first while loop prints numbers from 1 to 10.
2. The second part asks the user to enter a positive integer.
3. Input validation ensures that only positive integers are accepted.
4. If the user enters a negative number, zero, or non-integer value, an appropriate error message is displayed.
5. If the input is valid, the program prints numbers from 1 up to the entered number using a while loop.
